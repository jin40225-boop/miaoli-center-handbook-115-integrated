    第五篇 業務流程與專案 SOP¶

    專家到宅服務標準作業流程¶

    專家到宅服務係針對一般個案管理難以單靠主責社工處理，且案家同時具有多重、複雜或高風險議題時所啟動之支持性服務。其目的不在取代個案管理，而是在既有個案管理基礎上，額外連結適切之專業人員進入案家，協助處理心理、家庭、居家環境、法律、財務、社會適應或其他特定議題，使主責社工能在專業團隊支持下提供更完整之服務。 本中心目前可運用之專業支持類型，包含但不限於心理諮商、家庭支持、社會適應指導、居家整理、法律與經濟諮詢等。專家到宅服務應以個案需求為核心，由主責社工先完成評估、與督導討論後啟動，並依本中心既定核銷與資料留存規範辦理。

    二、步驟／流程¶

    1. 先判斷是否需要啟動專家到宅¶

    當個案或家庭出現以下情形之一時，可優先評估是否啟動專家到宅： - 1. 主責社工已持續提供服務，但關鍵問題仍無法突破。 - 2. 案家涉及多重需求，需借重其他專業角色共同介入。 - 3. 家庭具有高度照顧壓力、雙老、高照顧負荷、明顯家庭衝突或生活功能失衡。 - 4. 個案因行動受限、交通不便、精神或情緒因素，不易外出使用專業服務。 - 5. 督導評估認為需要透過外部專業協助，以提升處遇品質或降低風險。

    2. 主責社工先完成初步評估¶

    啟動前，主責社工應先整理以下重點： - 1. 案主與家庭目前面臨的主要問題。 - 2. 目前已提供過哪些服務、效果如何。 - 3. 為何現有個案管理不足以回應需求。 - 4. 本次希望透過專家到宅達成的目標。 - 5. 預計需要哪一類專業人員介入。

    3. 務必先與督導討論並取得同意¶

    專家到宅服務不得由主責社工自行決定逕行啟動。啟動前應： - 1. 向督導說明個案現況、需求與初步評估。 - 2. 釐清是否符合本中心目前可使用之專家到宅條件。 - 3. 確認本次由哪類專業人員進場最合適。 - 4. 確認可用時數、執行方式與後續核銷規範。

    4. 準備轉介資料與必要附件¶

    取得督導同意後，主責社工應依本中心規範準備轉介資料，至少包括： - 1. 個案基本資料。 - 2. 問題概述與需求說明。 - 3. 已有服務歷程摘要。 - 4. 本次轉介目的與期待目標。 - 5. 其他督導要求補充之資料。 若中心另有既定之轉介單、公版表格或附件，應依最新版本填寫，不得自行簡化或省略重要欄位。

    5. 安排到宅服務執行¶

    專業人員確認後，由主責社工協助安排： - 1. 與案家說明服務目的、內容與進行方式。 - 2. 確認案家可配合之日期、時段與地點。 - 3. 視需要與專業人員進行事前溝通。 - 4. 若屬高風險或需跨專業合作案件，可由主責社工陪同共同到宅。

    6. 服務過程應留存必要紀錄¶

    到宅服務執行後，應依中心規範留存以下資料： - 1. 服務紀錄或摘要。 - 2. 現場照片（若規範要求）。 - 3. 專業人員提供之回饋、服務意見或服務成果。 - 4. 主責社工後續追蹤與處遇安排。

    7. 完成後辦理送審與核銷¶

    服務完成後，主責社工應注意： - 1. 將照片與轉介單上傳雲端送審。 - 2. 當月填報服務時數控管表。 - 3. 若有領據，須於當月繳回行政（黃梅）辦理核銷。 - 4. 若服務需續用，應再與督導討論是否延續或調整介入方式。

    三、重要知識或判斷原則¶

    1. 專家到宅服務不是一般資源轉介的替代品¶

    若個案可以透過既有公費、一般福利或常態服務解決需求，原則上仍應先使用既有制度性資源。專家到宅主要用於一般服務不足、案家使用困難或需要跨專業深入介入的情況。

    2. 啟動前一定要先有主責社工評估¶

    專家到宅不是「案家想要就申請」，而是必須先經主責社工評估與督導同意。若沒有清楚的問題定義與服務目標，即使安排專家進場，也容易造成服務分散或效果不明。

    3. 服務目標要具體¶

    提出申請時，不宜只寫「需要協助」「需要支持」等模糊語句，而應寫明： - 需要處理的核心問題是什麼。 - 預計透過哪一類專業介入。 - 完成後希望看到哪種改變或改善。

    4. 主責社工仍負責後續整合¶

    即使已有專業人員到宅，個案主責仍在中心社工。後續資源連結、服務整合、追蹤與紀錄，仍須由主責社工負責統整與延續。

    四、注意事項¶

    • 未經督導同意，不得自行啟動專家到宅。
    • 啟動前應確認本案是否已先使用可近之公費或常態資源。
    • 轉介資料應具體、清楚，避免只寫概略描述。
    • 服務完成後，照片、轉介單、時數控管表與領據均須依規定時間完成。
    • 領據如逾期未繳，可能影響後續核銷與經費處理。
    • 若屬高風險案件，應視情況安排主責社工共同到場，避免專業人員單獨承擔風險。
    • 到宅服務若涉及敏感家庭議題、隱私或安全問題，應事先評估場域風險與案家配合情形。

    五、表單／系統／雲端位置¶

    • 專家到宅服務相關轉介單：依中心最新公版辦理。
    • 服務時數控管表：依中心當年度核銷資料夾辦理。
    • 照片與轉介單送審位置：依專家到宅服務雲端資料夾上傳。
    • 領據繳交窗口：行政黃梅。

    專家到宅服務標準作業流程¶

    一、目的¶

    本節整理苗栗縣身心障礙者服務中心（第一區）115 年度專家到宅服務之啟動時機、服務流程、角色分工、表單、核銷與控管方式，使主責社工於評估個案需求時，能清楚判斷何時可使用此項資源、如何安排專業人員進入案家，以及後續如何追蹤與管理。

    二、服務定位¶

    專家到宅服務屬於中心正式之個案處遇與家庭支持資源，不是臨時額外協助，也不是單純的外聘講師或個別諮詢。其核心精神為： - 提升服務可近性 - 以跨專業協作方式進入案家 - 協助處理一般社工訪視難以單獨回應的特定議題 - 讓服務在真實家庭情境中進行觀察、介入與調整 中心推動此模式，是因為部分身心障礙者及家庭因身體功能受限、心理困擾、照顧負荷高、交通不便或家庭支持不足，難以穩定外出接受服務，因此需由主責社工結合心理、醫護、職能、物理治療、語言治療、法律等專業資源，共同進入家庭。

    三、適用情形¶

    主責社工於評估過程中，如發現服務對象或照顧者有下列需求，可評估使用專家到宅服務： - 心理輔導需求 - 照護與健康促進需求 - 功能復健或生活適應需求 - 家庭關係與互動議題 - 法律諮詢需求 - 其他需跨專業共同介入之困難議題 尤其當個案具有以下特徵時，更可優先納入考量： - 外出接受服務困難 - 雙老家庭或高照顧負荷家庭 - 交通可近性不足 - 服務需求複雜，單靠一般社工訪視較難深入處理 - 需要在家庭現場觀察功能、互動與環境因素

    四、啟動前原則¶

    1. 務必先與督導討論¶

    專家到宅服務屬正式處遇資源，主責社工在啟動前，務必先與督導討論並取得同意，不宜自行直接聯繫專業人員進案。

    2. 先釐清要解決的核心問題¶

    主責社工於提出需求前，應先整理： - 個案目前處境 - 家庭主要困難 - 已嘗試過哪些處遇 - 為何需要專家到宅 - 希望專業人員協助的方向為何

    3. 先評估是否適合到宅¶

    並非所有個案都適合立即安排。若案家拒絕、風險過高未先穩定、或目的仍過於模糊，應先完成前置評估與關係建立。

    五、服務流程¶

    第一階段：初次接案與關係建立¶

    主責社工接案後，若於評估過程中發現服務對象或照顧者有特定專業需求，得依個案議題與實際需要，連結相關專業人員介入，安排後續共同家訪或到宅服務。 此階段重點為： - 1. 主責社工先完成基本評估。 - 2. 與督導確認可啟動專家到宅。 - 3. 初步確認需要何種類型專業人員。 - 4. 向案家說明專家到宅的目的與可能方式。

    第二階段：第一次家訪－共同初評¶

    由社工與相關專業人員共同進入案家。原則上由社工主導會談，專業人員協助觀察與評估服務對象的身心狀況、家庭互動、功能狀態與需求方向。 本次家訪主要任務： - 1. 與案家建立信任與合作基礎。 - 2. 由社工說明中心角色與此次到宅目的。 - 3. 由專業人員進行初步觀察與判斷。 - 4. 於現場討論是否需要後續持續介入。 - 5. 如確認有需要，於當次完成下次約訪時間。

    第三階段：第二次家訪－角色分工與合作（視需要安排）¶

    若確認需持續進行，第二次家訪原則上由專業人員主導、社工為輔。社工在會談中需向案主與家屬清楚說明分工： - 專業人員：依個案議題提供技術性與功能性服務，例如心理輔導、復健訓練、健康促進、法律建議等。 - 主責社工：負責正式與非正式資源連結、家庭支持、整體服務統籌與後續追蹤。

    第四階段：持續到宅服務與追蹤¶

    專業人員依約定時間定期前往案家提供服務，並於每次服務後與主責社工討論服務狀況與成效。 主責社工需同步進行： - 依個案分級原則安排追蹤訪視頻率 - 持續了解服務對象與家庭對服務的反應 - 補強正式與非正式資源連結 - 視需要調整整體服務方向

    第五階段：每六次服務進行檢視¶

    主責社工每累積六次服務，需與專業人員共同檢視服務進展，討論是否： - 持續進行 - 調整服務方向 - 更換專業介入重點 - 準備結束到宅服務

    第六階段：結案前通知與後續銜接¶

    如擬結案，專業人員應於結案前三次通知社工，以利主責社工預先安排後續追蹤、資源銜接與家庭說明，不宜在服務結束當次才臨時告知。

    六、跨專業督導與檢討機制¶

    中心辦理督導會議時，視服務需求或個案情形，得邀請參與到宅服務之專業人員共同與會，進行： - 個案討論 - 服務經驗交流 - 介入困難整理 - 後續處遇方向修正 此作法的目的，不只是回顧個案，更是提升跨專業合作默契與整體服務品質。

    七、表單與文件管理¶

    為確保服務紀錄完整性及經費運用合規性，所有專家到宅服務均應依規填寫並留存下列表單：

    1. 專業人員到宅服務轉介單¶

    • 由主責社工依個案議題轉介相關專業人員使用。
    • 轉介後須留存副本。
    • 並上傳至個案管理系統。

    2. 服務紀錄表¶

    • 每次到宅服務後均應填寫。
    • 須於訪視後當月內完成填寫與繳交。
    • 由主責社工負責彙整、存檔，並上傳至個案管理系統留存。
    • 逾期未繳交者，暫緩核銷作業。

    3. 核銷領據¶

    • 每次服務結束後，專業人員應上傳服務紀錄及領據。
    • 供後續核銷使用。
    • 主責社工應確認資料是否完整，再交由行政流程續辦。

    4. 專家到宅服務控管表¶

    • 由主責社工彙整專業人員服務紀錄及核銷資料。
    • 登錄於控管表。
    • 供後續查核、分析與年度成果統計使用。

    八、費用與核銷原則¶

    1. 服務費標準¶

    專家到宅服務費以每小時新臺幣 2,500 元計。

    2. 行政管理費說明¶

    於說明轉介及簽約前，須事先告知專業人員，中心將扣除 15% 行政管理費用。

    3. 撥款原則¶

    服務紀錄經核對無誤後，於核銷作業完成後撥款，由行政人員依流程執行撥付。

    4. 核銷時應注意¶

    • 紀錄與領據要同時完整
    • 逾期紀錄將影響核銷
    • 主責社工不可只口頭說有服務，未補正文件即送核
    • 若資料不完整，應先補正再進入撥款流程

    九、主責社工的核心任務¶

    專家到宅服務雖有專業人員參與，但主責社工並未退出個案工作，反而更需要做好整合與統籌。主責社工至少應完成以下任務： - 1. 辨識需求並提出轉介 - 2. 與督導確認啟動 - 3. 協助安排第一次共同家訪 - 4. 向案家說明服務目的與分工 - 5. 追蹤服務狀況與家庭反應 - 6. 彙整文件、控管表與核銷資料 - 7. 每六次服務與專業人員共同檢視成效 - 8. 結案前做好後續資源銜接

    十、注意事項¶

    • 啟動前一定要先與督導討論，不宜自行先約專家。
    • 社工應避免把專家到宅視為「轉出去就結束」，仍需持續統籌整體服務。
    • 若案家需求與原先預估不同，應及時回到督導與專業人員共同修正方向。
    • 每次服務後的紀錄、領據與控管表都要追，否則容易造成核銷延誤。
    • 若服務擬停止或結束，不可等到最後一次才處理銜接。

    疑似或社區精神病人照護優化計畫轉介¶

    115 年度疑似或社區精神病人照護優化計畫依據「強化社會安全網 2.0（115-119 年）」推動，目的在於強化社區心理衛生服務，透過早期發現、即時介入與持續追蹤，提供疑似或確診精神病人連續性照護。 本篇整理申請與轉介作業流程，供主責社工判斷何時需啟動轉介、應填哪些附件，以及後續聯繫方式。

    二、為何需要轉介¶

    當社區中出現疑似精神症狀或精神醫療風險時，需透過轉介協助就醫評估、緊急處置與社區外展照護。 可評估轉介的情境包括： - 出現疑似精神症狀，例如妄想、幻覺等 - 有自傷或傷人風險 - 疑似精神病人被護送就醫但未住院 - 個案已自行就醫，包含門診或急診 - 醫師建議住院但個案不願住院

    三、收案條件¶

    收案條件須符合以下其中一類：

    第 1 類¶

    網絡轉介或社區民眾通知之疑似精神病個案，經評估需精神醫療處置者。

    第 2 類¶

    疑似精神病人被護送就醫但未住院者。

    第 3 類¶

    個案自行就醫，包含門診或急診，醫師建議住院但不願意住院者。

    四、服務對象資格限制¶

    本計畫不得與下列服務重疊： - 居家治療 - 精神復健機構，包含日間型及住宿型 - 精神護理之家服務 若個案已由社區心理衛生中心提供個案管理服務，且經評估仍具明確精神醫療服務需求，需與主辦或協辦醫院共同討論個案處遇。 可共同討論或合作的單位包含： - 大千南勢醫院 - 苗栗醫院 - 為恭醫院

    五、轉介流程¶

      1. 發現疑似精神症狀或精神醫療風險。
      1. 初步確認個案是否符合第 1、2、3 類收案條件。
      1. 檢查是否與居家治療、精神復健機構或精神護理之家服務重疊。
      1. 若已由社區心理衛生中心服務，先確認是否仍有明確精神醫療需求。
      1. 填寫對應轉介附件與服務同意文件。
      1. 將表單以信件寄送至指定信箱。
      1. 於個案紀錄中記錄轉介原因、寄送日期、聯絡窗口與後續回覆。

    精神病人照護優化計畫轉介流程¶

    本流程係針對具精神症狀、疑似急性風險或已有明顯醫療照護需求之個案，協助主責社工依既定收案條件與轉介規範，判斷是否轉介精神病人照護優化計畫。此類案件常涉及醫療、心理衛生、家庭照顧與危機管理等多面向議題，故需依既定標準辦理，避免重複轉介、錯誤轉介或漏附必要文件，影響後續收案與服務銜接。

    二、步驟／流程¶

    1. 先判斷是否屬於可轉介對象¶

    目前可評估之主要對象有三類： - 1. 網絡或長照轉介，且個案具精神症狀並需醫療處置者。 - 2. 經緊急送醫後未住院，但仍需後續追蹤與連結者。 - 3. 門診或急診醫師評估建議住院，但個案無住院意願者。

    2. 先排除不得轉介對象¶

    若個案目前已接受以下任一服務，原則上不再重複轉介至本計畫： - 1. 居家治療。 - 2. 精神復健機構服務。 - 3. 精神護理之家服務。 若主責社工對個案現況有疑義，應先與督導討論，確認目前服務體系與主責系統，避免重複收案。

    3. 與督導確認是否正式啟動轉介¶

    判斷符合收案條件後，主責社工應： - 1. 向督導摘要說明個案精神狀況、家庭支持與目前風險。 - 2. 確認是否已先啟動必要危機處理。 - 3. 確認應由哪一類轉介途徑送件。 - 4. 確認本案是否需與心理衛生中心同步聯繫。

    4. 與心理衛生中心及合作醫院銜接¶

    此類案件與心理衛生中心關聯度高，必要時需與大千南勢、苗栗或為恭醫院等合作單位進行危機處理與連結。主責社工應確認： - 1. 個案目前是否已有醫療接觸。 - 2. 是否需補強急性處遇。 - 3. 是否已建立與心理衛生中心之聯繫窗口。 - 4. 轉介後由誰持續追蹤。

    5. 備齊文件後送件¶

    依個案類別備齊相關文件，至少應包括： - 1. 對應轉介單。 - 2. 篩檢表（附件 3）。 - 3. 檢傷量表（附件 4）。 - 4. 服務同意書（附件 6）。 服務同意書為必要文件，未取得個案或家屬知情同意，不得逕行送件。

    6. 依規定送件¶

    文件完成後，依計畫規範送件，可使用下列方式： - 電子郵件：mlh5430@ems.miaoli.gov.tw - 傳真：037-374977 送件後應自行留存副本，並於案件紀錄中註記送件日期、送件方式與後續回應情形。

    7. 後續追蹤¶

    送件後主責社工仍應： - 1. 追蹤是否成功收案。 - 2. 若未收案，確認原因並補件或改採其他處遇方式。 - 3. 持續記錄與家庭、醫療與心理衛生系統之互動過程。 - 4. 視情況調整個案管理計畫。

    三、重要知識或判斷原則¶

    1. 此類案件重點在危機與醫療銜接¶

    若個案已有精神症狀或急性危機，社工不能只做一般資源轉介，應優先評估其醫療需求、急性風險與家庭支持狀態，必要時先啟動心理衛生或醫療系統。

    2. 先排除重複服務¶

    個案若已接受居家治療、精神復健機構或精神護理之家服務，表示已有特定主責系統，若未先釐清即重複轉介，容易造成角色混亂與服務重疊。

    3. 服務同意不可省略¶

    精神病人照護優化計畫涉及後續持續服務與追蹤，服務同意書是必要文件，亦是維持服務正當性與保障個案權益的重要依據。

    四、注意事項¶

    • 不得只因個案有精神疾病診斷，即逕認為一定符合本計畫轉介條件。
    • 送件前應先確認是否有既有服務系統正在處理。
    • 檢附文件須完整，尤其服務同意書不得漏附。
    • 若個案有明顯急性危險、自傷傷人風險或明顯失序，應優先處理危機，不宜僅停留在一般送件程序。
    • 送件後仍須追蹤，不可送出即視為結案。
    • 後續連結結果應回寫於案件處理紀錄表或相關個案紀錄中。

    精神病人照護優化計畫轉介流程¶

    一、目的¶

    本節整理精神病人照護優化計畫之適用對象、排除原則、轉介步驟、應備文件、送件方式與後續追蹤機制，協助主責社工在個案涉及精神症狀、急性危機或需精神醫療銜接時，有一致的轉介作法。

    二、計畫定位¶

    精神病人照護優化計畫屬於精神健康與社區支持的重要轉介資源。 其目的不是單純把精神相關問題「轉出去」，而是讓主責社工在辨識需求後，能透過既定合作網絡，把需要醫療處置、危機介入或持續追蹤的個案，連結到更合適的精神照護系統。

    三、收案評估對象¶

    目前可評估轉介之對象，主要分為三類：

    1. 網絡或長照轉介¶

    由其他網絡單位或長照系統轉介，且個案具精神症狀並需醫療處置者。

    2. 急送醫未住院者¶

    經緊急送醫後，醫師評估無須住院，但後續仍需精神健康追蹤與照護銜接者。

    3. 醫師建議住院但個案拒絕者¶

    經門診或急診醫師建議住院，但個案無意願住院，後續仍有明顯風險與追蹤需要者。

    四、排除原則¶

    如個案已接受下列任一服務，原則上不再重複進入本計畫： - 居家治療 - 精神復健機構服務 - 精神護理之家服務 因此轉介前，務必先確認個案目前是否已在其他精神醫療或復健照護系統中接受正式服務。

    五、轉介前評估重點¶

    主責社工在提出轉介前，至少應先確認： - 1. 個案是否符合三類收案對象之一。 - 2. 是否已有現行精神醫療、居家治療或精神復健服務。 - 3. 目前最主要的精神健康風險是什麼。 - 4. 是否已有急診、門診、送醫或醫師建議住院紀錄。 - 5. 家屬是否知情並同意進一步轉介。 - 6. 目前是否有立即危機，需要先處理安全問題再送件。

    六、115年度完整轉介流程¶

    依現行規範，可整理為六步驟：

    步驟一：確認是否符合收案條件¶

    先確認個案是否屬三類收案評估對象之一，並同步排除不符條件或重複服務情形。

    步驟二：確認未同時接受排除性服務¶

    再次確認個案未同時接受居家治療、精神復健機構或精神護理之家服務，避免重複轉介或資源錯置。

    步驟三：必要時與心理衛生中心及合作醫院連結¶

    若個案涉及危機評估、送醫或醫療銜接，需與心理衛生中心及合作醫院進行危機處理與連結。現行合作醫療端包括： - 大千南勢 - 苗栗相關合作醫院 - 為恭醫院

    步驟四：備齊文件¶

    依個案類別備齊轉介資料，至少包括： - 對應轉介單 - 篩檢表（附件 3） - 檢傷量表（附件 4） - 服務同意書（附件 6） 其中服務同意書為必要文件，務必確認個案或家屬已知情同意。

    步驟五：完成送件¶

    資料確認完整後，依現行方式送出。現行送件資訊為： - 電子信箱：mlh5430@ems.miaoli.gov.tw - 傳真：037-374977

    步驟六：追蹤回覆與後續銜接¶

    送件後，主責社工仍需持續追蹤： - 是否已收件 - 是否需補件 - 是否已啟動後續處理 - 案家是否知悉下一步 - 是否需要中心持續協助危機穩定與資源連結

    七、主責社工的角色¶

    精神病人照護優化計畫不是送件即完成。主責社工至少應負責： - 1. 辨識是否適合轉介 - 2. 確認收案與排除條件 - 3. 蒐集送件所需文件 - 4. 與案家說明轉介目的與同意內容 - 5. 完成送件並追蹤回覆 - 6. 必要時與督導、心理衛生中心、合作醫院共同討論 - 7. 將後續結果與追蹤內容納入中心服務紀錄

    八、與中心個案服務的關係¶

    即使已完成轉介，中心仍須視個案狀況持續提供必要支持。 主責社工不可因轉介完成，就完全退出後續追蹤。尤其若個案同時有下列情況，更應持續關注： - 家庭照顧功能不穩 - 情緒與行為風險高 - 近期有送醫或住院建議 - 家屬對精神疾病照護理解不足 - 有拒醫、拒服藥或拒絕住院情形

    九、注意事項¶

    • 先確認是否符合三類收案條件，再進入送件作業。
    • 已接受居家治療、精神復健機構或精神護理之家服務者，原則上不重複送件。
    • 服務同意書為必要文件，不可漏附。
    • 送件後要追，不可只寄出就結束。
    • 若個案有立即危機，應先處理安全與危機穩定，再談後續轉介程序。

    民間資源申請與社會福利資源分流¶

    個案服務過程中，常會遇到案家需求超出現行政府福利範圍，或雖有法定資源，但短期內難以即時到位之情況。此時，社工除需熟悉公部門法定福利外，也應具備民間資源申請與分流判斷能力，避免個案在等待期間出現服務落空、家庭壓力升高或風險惡化。 本篇重點在於協助社工先區分哪些需求應優先走政府法定資源，哪些需求可搭配民間資源補充，並在申請後持續追蹤，將結果如實記錄於案件處理紀錄中。

    二、步驟／流程¶

    1. 先確認個案需求屬性¶

    接到需求後，先區分目前主要需求偏向哪一類： - 1. 經濟支持或急難需求。 - 2. 物資補給。 - 3. 技能學習或職業訓練。 - 4. 心理諮商或心理支持。 - 5. 輔具、居家、照顧或長照相關需求。 - 6. 其他政府制度尚未完全涵蓋之特殊需求。

    2. 先看是否已有公費或法定資源可用¶

    原則上政府補助優先。社工應先確認： - 1. 個案是否已申請相關法定福利。 - 2. 是否符合政府補助資格但尚未申請。 - 3. 是否僅因資料不足、程序不熟或等待時間過長，導致服務尚未進入。 若法定資源仍可使用，不宜直接跳過公部門制度而只申請民間資源。

    3. 依需求進行分流¶

    分流一：法定福利與交通、經濟補助¶

    適用例如： - 身心障礙者專用停車位識別證 - 復康巴士 - 生活補助 - 醫療費用補助 - 房屋或停車位租金補貼

    分流二：居家、輔具與長照類¶

    適用例如： - 居家護理或居家復健 - 送餐服務 - 輔具補助 - 長照服務 - 生活重建、心理重建 - 社區居住 - 日間或夜間照顧

    分流三：家庭支持與照顧者支持¶

    適用例如： - 臨時照顧 - 短期照顧 - 照顧者支持服務 - 照顧者訓練 - 其他可減輕家庭照顧壓力之支持性資源

    4. 若法定資源不足，再評估民間資源¶

    若個案已有法定資源但仍有缺口，可再評估民間資源，常見類型包括： - 1. 急難救助金。 - 2. 物資補給。 - 3. 技能學習與職業訓練。 - 4. 心理諮商資源。 - 5. 身障輔具補充。

    5. 申請前與督導討論確認¶

    民間資源申請前，至少應確認： - 1. 本案是否已先走政府資源。 - 2. 民間資源申請目標是否明確。 - 3. 是否符合民間單位收案資格。 - 4. 是否已有其他單位重複申請。 - 5. 是否需同步通知主責督導或相關網絡。

    6. 送件與追蹤¶

    送件後，社工應： - 1. 留存送件紀錄。 - 2. 記錄申請日期、單位、項目與聯繫窗口。 - 3. 每月追蹤進度。 - 4. 將結果寫入案件處理紀錄表。

    三、重要知識或判斷原則¶

    1. 政府補助優先¶

    民間資源不是第一順位替代方案。原則上應先確認法定福利可否使用，再評估民間資源補充，以免資源使用失衡或重複。

    2. 分流的目的不是把案家推走¶

    分流不是拒絕處理，而是讓需求進入最適合的系統。社工應協助案家理解每一類資源的性質、限制與申請方式，而不是只告知「這不是我們的」。

    3. 申請後仍需追蹤¶

    很多資源申請不是送出就完成，若沒有後續追蹤，案家仍可能因缺件、資格不符、窗口不明或流程停滯而無法真正取得服務。

    四、注意事項¶

    • 申請民間資源前，務必先確認已檢視法定資源。
    • 需避免同一需求由多個單位重複申請而造成混亂。
    • 若申請被退件或不符資格，應主動協助案家改走其他替代方案。
    • 所有進度與結果都應回寫於案件處理紀錄表。
    • 若屬高急迫性需求，切勿等待單一資源結果，應同步思考替代方案。

    民間資源申請與社會福利資源分流¶

    一、目的¶

    本節整理當個案或家庭出現福利需求時，主責社工應如何先做法定福利資源分流，再評估是否需要申請民間資源，避免直接跳過政府資源、造成資源錯置或紀錄不完整。

    二、基本原則¶

    民間資源申請的核心原則為： - 1. 政府補助優先 - 2. 依法定福利先分流 - 3. 民間資源作為補充，不取代法定資源 - 4. 每月追蹤進度並記錄於案件處理紀錄表 也就是說，遇到需求時，第一步不是先找哪個民間單位，而是先判斷這項需求本身屬於哪一類社會福利服務。

    三、何時需要評估民間資源¶

    當個案有政府資源以外的需求，或雖已使用政府資源但仍有缺口時，可評估連結民間資源。常見需求例如： - 急難救助金 - 物資補給 - 技能學習與職業訓練 - 心理諮商資源 - 身障輔具補充 - 臨時性支持或過渡性協助

    四、民間資源申請的基本流程¶

    步驟一：先確認需求內容¶

    先釐清案家真正需要的是： - 經濟補助 - 交通協助 - 生活照顧 - 輔具 - 長照 - 家庭照顧者支持 - 心理支持 - 就學、就業或日照銜接 - 其他民間補充性資源

    步驟二：先確認是否已申請公費資源¶

    在評估民間資源前，應先確認： - 是否已有相對應政府補助或法定福利可申請 - 是否已提出申請 - 是否不符資格 - 是否仍在候缺或等待通知 - 是否雖已使用，但需求仍未被滿足

    步驟三：依福利需求先做社會福利資源分流¶

    目前可先依三類分流理解：

    分流一：交通與經濟補助¶

    常見包括： - 身心障礙者專用停車位識別證 - 復康巴士 - 生活補助費 - 醫療費用補助 - 房屋租金補貼 - 停車位租金補貼等

    分流二：居家、輔具與長照¶

    常見包括： - 居家護理 - 居家復健 - 送餐服務 - 輔具補助 - 長照服務 - 生活重建 - 心理重建 - 社區居住 - 日間或夜間照顧等

    分流三：家庭照顧者支持與其他支持性服務¶

    常見包括： - 家庭照顧者支持服務 - 臨時及短期照顧 - 照顧者支持課程與訓練 - 其他正式福利體系下可申請之支持服務

    步驟四：評估是否需要補充民間資源¶

    若經法定福利分流後仍有缺口，可再評估民間機構申請資格。此時應考量： - 是否確有補充必要 - 是否有明確缺口 - 民間資源是否符合該家庭情況 - 是否有重複申請或重複補助疑慮

    步驟五：與督導討論確認¶

    在正式啟動民間資源申請前，原則上應先與督導討論確認，尤其是： - 補助金額較高 - 需求判斷有爭議 - 案家條件複雜 - 同時涉及多套資源 - 有可能與公費資源重疊時

    步驟六：送件、追蹤與紀錄¶

    送件後需每月追蹤進度，並將結果記錄於案件處理紀錄表，包括： - 是否已送件 - 是否需補件 - 核定結果 - 使用情形 - 問題是否獲得改善 - 若未通過，是否需改尋其他資源

    五、主責社工在分流中的角色¶

    主責社工在資源分流與民間申請中的角色，不只是幫忙找單位，而是要先做判斷與整合，包括： - 1. 釐清需求本質 - 2. 區分法定福利與補充性民間資源 - 3. 先確認公費資源使用狀況 - 4. 評估缺口是否真需要民間資源補充 - 5. 與督導討論 - 6. 後續追蹤與記錄成效

    六、常見錯誤¶

    1. 一有需求就直接找民間單位¶

    這樣容易跳過原本就可使用的法定福利，造成資源分流錯亂。

    2. 只送件不追蹤¶

    若沒有後續追蹤，就無法知道資源是否核定、是否真的改善問題。

    3. 只看「有沒有補助」，沒看「問題有沒有解決」¶

    有些家庭即使已連結資源，實際困難仍未被解決，後續仍需再調整資源組合。

    4. 沒有留下紀錄¶

    民間資源使用情形、進度與結果都應納入案件處理紀錄表，不可只靠口頭記憶。

    七、與分級的關係¶

    若個案在資源使用面向出現明顯困難，例如： - 不會使用社會資源 - 資源已連結但候缺 - 資源已使用但問題未解決 - 因環境限制資源使用困難 則除進行資源分流外，也應同步回到案件分級判斷，確認是否需維持較高服務強度。

    八、注意事項¶

    • 政府補助優先，是此篇最核心原則。
    • 民間資源是補充，不是直接跳過法定福利。
    • 每月追蹤進度，結果均需記錄於案件處理紀錄表。
    • 若需求複雜或補助重疊疑義高，應先與督導討論。
    • 評估資源時，不只看是否核定，也要看是否真的改善案家問題。

    通霄家託宣導與教育生涯轉銜處理¶

    本篇主要整理與通霄家託宣導、生涯轉銜及教育轉銜相關之工作處理原則。此類案件與一般個案管理不同之處，在於其往往涉及就學、就業、日間照護或其他生活階段轉換，因此除了一般關懷服務外，更需要處理跨系統轉銜、紀錄同步與服務強度維持等事項。

    二、步驟／流程¶

    1. 先判斷個案是否涉及轉銜需求¶

    以下情形可優先評估啟動生涯或教育轉銜： - 1. 個案有就學需求。 - 2. 個案有就業需求。 - 3. 個案有日間照護需求。 - 4. 個案即將從原服務體系畢業、離開或轉入另一系統。 - 5. 個案目前雖未使用服務，但已明顯進入需重新規劃生活安排之階段。

    2. 若非既有處遇案，應先開新案¶

    若個案目前並非中心既有處遇案件，而後續需正式處理其轉銜服務，原則上應以「開新案」方式辦理，不宜只做零散諮詢或口頭協助。

    3. 轉銜案件應維持一定服務強度¶

    轉銜案件原則上不可一開始就以四級處理。依中心現行做法： - 1. 轉銜案件最少須以三級或二級開案。 - 2. 須經至少 2 次或 3 次服務後，方可視狀況評估是否轉為四級。 - 3. 教育轉銜案件另有既定追蹤頻率與紀錄要求時，應從其規定辦理。

    4. 連結所需資源¶

    依個案需求，連結適切之服務體系，例如： - 1. 教育系統資源。 - 2. 生涯轉銜平台或相關單位。 - 3. 就學、就業、就養或日間照顧資源。 - 4. 家託、家庭支持或社區照顧相關資源。 - 5. 督導認為應同步納入之其他網絡。

    5. 紀錄需同步¶

    主責社工須定期將服務紀錄轉貼至生涯轉銜系統，確保： - 1. 個管系統有紀錄。 - 2. 轉銜系統也有紀錄。 - 3. 兩邊資訊內容一致，不可一邊有做、一邊空白。

    三、重要知識或判斷原則¶

    1. 轉銜案件不是一般穩定個案¶

    轉銜本質上就是生活階段轉換，若處理過輕，容易在就學、就業、照顧安排或服務銜接上產生落差，因此不宜一開始就以最輕度服務對待。

    2. 開案不是形式，而是確保可持續追蹤¶

    若個案已明確需要中心介入轉銜，正式開案有助於後續追蹤、系統登打、督導查核與跨網絡協作，不宜只停留在非正式聯繫。

    3. 雙系統紀錄很重要¶

    若個案同時涉及個管系統與轉銜系統，兩邊都需有紀錄，否則後續容易出現工作斷層、責任不清或查核時資料不一致。

    四、注意事項¶

    • 若個案具明確轉銜需求，不宜直接四級開案。
    • 非既有處遇案者，需確認是否應開新案。
    • 紀錄不得只留在單一系統。
    • 轉銜案件常伴隨家庭壓力、資訊不清與資源落差，應同步評估家庭支持需求。
    • 若個案需求跨教育、社政、就業等多系統，應及早與督導討論合作方式。

    通霄家託宣導與教育生涯轉銜處理¶

    一、目的¶

    本節整理通霄家託相關宣導合作，以及個案涉及就學、就業或日間照護需求時，中心如何啟動教育生涯轉銜服務，協助主責社工在面對轉銜案件時，有一致的判斷與紀錄方式。

    二、主題定位¶

    本篇其實包含兩個彼此連動的工作面向： - 1. 通霄家託相關宣導與合作事項 - 2. 教育生涯轉銜服務啟動與追蹤處理 之所以放在同一篇，是因為實務上這類案件常同時涉及社區宣導、家屬理解、服務引介與後續轉銜追蹤。

    三、通霄家託宣導原則¶

    通霄家託相關業務流程，原則上依最新 SOP 辦理。 主責社工在實務上應先確認： - 此次工作屬單純宣導、服務介紹，還是已進入個案評估與轉銜需求 - 是否需後續開案 - 是否需引導家屬了解日間照護、就學、就業或其他支持資源 - 是否需轉入正式生涯轉銜服務追蹤

    四、何時需要啟動教育生涯轉銜服務¶

    當個案具有下列需求時，應啟動生涯轉銜服務評估： - 就學需求 - 就業需求 - 日間照護需求 - 畢業後未升學、未就業之追蹤需要 - 原有服務已不符合現階段生活安排，需要轉換支持系統

    五、轉銜案件的基本原則¶

    1. 非現有處遇案，應以開新案方式辦理¶

    若個案並非中心既有處遇案，原則上須以開新案方式處理，不宜只做口頭關懷或單次紀錄後即結束。

    2. 轉銜案件最少應列三級或二級¶

    轉銜案件原則上最少需以三級，必要時以二級開案，不宜直接列四級。

    3. 不可太快降為四級¶

    轉銜案件必須經過至少 2 次或 3 次服務 後，視實際處遇狀況，方可評估是否轉為四級。 也就是說，不能剛開案、剛建資料或剛說明完資源，就立刻降為最低關懷強度。

    六、主責社工的工作內容¶

    1. 釐清轉銜需求¶

    先了解目前需求究竟是： - 升學銜接 - 就業規劃 - 日照或日間照護安排 - 家屬對未來生活安排的不安 - 個案本人無法表達，但實際已有生活空窗

    2. 盤點現有支持系統¶

    確認目前是否已有： - 學校資源 - 就服資源 - 日照服務 - 家屬支持 - 其他網絡追蹤

    3. 評估是否需正式開案¶

    若已明確需要中心持續追蹤、資源連結與服務統籌，就應正式開案，不宜只停留在宣導或單次諮詢。

    4. 進行至少基本服務次數¶

    開案後，應依實際情形完成至少 2 次或 3 次服務，包含面訪、電訪、資源說明、連結與追蹤，之後再評估是否能降低服務強度。

    七、系統紀錄原則¶

    1. 個管系統要有紀錄¶

    主責社工應依中心一般個案服務規則，將服務內容記錄於個案管理系統。

    2. 生涯轉銜系統也要同步¶

    主責社工須定期將服務紀錄轉貼至生涯轉銜系統內，確保： - 個案管理系統有紀錄 - 生涯轉銜系統也有紀錄 兩邊都要有，不能只留在其中一個系統。

    八、為什麼轉銜案件不能做得太輕¶

    轉銜案件表面上看似「只是未來安排」，但實際上常涉及： - 家庭對障礙者成年後生活的焦慮 - 個案就學與就業資源斷裂 - 畢業後生活節奏突然中斷 - 家庭照顧負荷增加 - 日間支持不足而退回家庭照顧 因此，轉銜案件若處理太輕、追蹤太少，容易在短期內形成家庭壓力與服務空窗。

    九、注意事項¶

    • 通霄家託業務流程仍以最新 SOP 為準。
    • 凡有就學、就業或日間照護需求者，應想到生涯轉銜服務。
    • 非現有處遇案，原則上應開新案。
    • 轉銜案件至少以三級或二級開案，不宜直接列四級。
    • 個管系統與生涯轉銜系統兩邊都要有紀錄。
    • 至少完成 2 次或 3 次服務後，再談是否可降為四級。

    社區宣導辦理 SOP¶

    社區宣導係本中心年度重要工作之一，目的在於提升民眾對身心障礙權益、福利資源、雙老家庭、高照顧負荷家庭、成人監護或輔助宣告等議題的認識，並在活動現場建立與社區、學校、網絡單位及一般民眾的接觸機會。宣導工作除現場口頭宣講外，亦涉及前期規劃、文宣準備、簽到與成果資料整理，故需依既定流程辦理。

    二、步驟／流程¶

    1. 前期確認¶

    主責人收到活動或宣導任務後，應先確認： - 1. 時間、地點與對象。 - 2. 本次宣導主題。 - 3. 現場設備與空間條件。 - 4. 是否需搭配簡報、講義、宣導品或簽到表。 - 5. 是否需事先與承辦單位、公文窗口或內部行政協調。

    2. 準備宣導資料¶

    主責人應準備： - 1. 宣導簡報。 - 2. 講義或 DM。 - 3. 簽到表。 - 4. 宣導品、手拿板、PVC 或其他視覺素材。 - 5. 現場所需筆電、簡報筆、麥克風、投影設備等。 宣導簡報原則上存放於： - \3-8身心障礙福利宣導活動

    3. 進行現場宣導¶

    宣導現場應： - 1. 依活動安排進行簡短而清楚的宣講。 - 2. 口頭宣講約 3 至 5 分鐘，若另有完整課程則依主辦單位流程辦理。 - 3. 向民眾說明中心功能、聯繫方式與可用資源。 - 4. 視主題提供相關福利或服務資訊。

    4. 完成簽到與現場紀錄¶

    活動當日應： - 1. 回收簽到表。 - 2. 拍攝至少 6 張可作成果佐證之照片。 - 3. 確認照片包含現場、人員互動、宣導內容或活動布置等有效畫面。

    5. 活動後資料整理¶

    活動後應： - 1. 於伺服器新增本次活動資料夾。 - 2. 上傳活動照片與簽到表。 - 3. 至宣導活動統計表填寫本次資料。 - 4. 若活動需成果報告，應依規定另行撰寫。

    三、重要知識或判斷原則¶

    1. 宣導不只是完成場次¶

    宣導真正重點是讓民眾理解身心障礙相關資訊與可用資源，因此現場講述應盡量清楚、貼近民眾情境，而不是只求完成場次。

    2. 成果資料要能作為佐證¶

    簽到表與照片不只是備查，而是後續成果報告、指標統計與核銷的重要依據，因此應於活動當日確實完成。

    3. 現場宣導內容應與主題一致¶

    不同主題之宣導內容應有所區分。例如雙老家庭、高照顧負荷、身心障礙權益或監護輔助宣告，各自有不同重點，避免一律使用相同講稿而失去宣導效果。

    四、注意事項¶

    • 宣導前要再次確認時間、地點與設備，避免現場臨時出問題。
    • 若使用簡報與投影，應事先測試檔案與設備。
    • 簽到表與照片務必於當日完成蒐集。
    • 現場若有人提出明確求助需求，應視情況引導至正式聯繫或後續服務，而非僅留在活動現場口頭回覆。
    • 宣導內容若涉及主管機關要求或年度指標，應配合當年度最新規範調整。

    社區宣導辦理SOP¶

    一、目的¶

    本節整理中心辦理社區宣導之完整流程，協助同仁從接洽、規劃、文宣準備、活動執行到成果整理，都有一致的基本作法。 本篇所稱社區宣導，包含實體宣講、擺攤、入校、社區場域宣導、系列座談會、活動結合式宣導等型態。

    二、宣導的工作定位¶

    社區宣導不是單純「去講一場」，而是中心正式業務的一部分。 其功能至少包括： - 傳遞身心障礙福利與服務資訊 - 提升社會大眾對身心障礙議題的理解 - 增加潛在服務對象與網絡對中心的辨識度 - 支持年度計畫中的宣導指標與成果要求 - 作為後續服務引介與資源銜接的入口

    三、宣導型態¶

    中心常見宣導型態可分為三類：

    1. 口頭宣導¶

    例如： - 社區宣講 - 校園宣導 - 網絡會議分享 - 活動現場簡報或口頭說明

    2. 文字與實體文宣¶

    例如： - DM - 文宣摺頁 - 宣導貼紙 - PVC 打卡板 - 手拿板 - 背板 - 立牌 - 其他活動陳列物

    3. 數位宣導¶

    例如： - Facebook 粉專圖文 - LINE 官方帳號訊息 - 活動宣傳圖卡 - 線上座談或報名資訊露出

    四、辦理前的整體流程¶

    社區宣導原則上可依下列節奏規劃：

    第一階段：確認宣導需求與主題（至少應提前 2 個月）¶

    先確認： - 這次宣導的主題是什麼 - 對象是誰 - 場地在哪裡 - 辦理型態是講座、擺攤、入校、座談或活動結合 - 是否屬年度既定場次或臨時新增需求 - 是否需要公文、報名、合作單位聯繫或講師邀請 如屬需對外聯繫、正式邀請或跨單位合作，應及早與督導討論確認方向。

    第二階段：確認合作窗口與行政需求（至少應提前 1.5 個月）¶

    此階段需確認： - 對外聯繫窗口 - 是否需發函 - 是否需確認講師或分享人 - 場地可用時間與設備 - 是否需報名表單 - 是否需簽到表、滿意度調查或成果統計

    第三階段：確認宣導內容與文宣形式（至少應提前 1 個月）¶

    此階段需開始準備： - 簡報 - 講綱 - 宣導品 - 文宣 - 場佈需求 - 線上宣傳圖文 - 活動當日流程表 若是擺攤型宣導，應特別確認現場是否需要： - 桌椅 - 背板 - 打卡板 - 手拿板 - 立牌 - 問卷 QR code - 音響或延長線

    第四階段：追蹤報名與資料補齊（至少應提前 2 週）¶

    此階段應確認： - 報名人數 - 合作單位是否已完成通知 - 文宣是否已送達 - 簡報是否定稿 - 簽到表、滿意度調查、統計表是否已準備完成 - 是否需提醒講師、主持人或支援同仁

    第五階段：活動前最後確認（提前 1 週至 3 天）¶

    至少要再確認： - 場地 - 設備 - 人力分工 - 茶水或便當 - 講師或主講者 - 拍照人員 - 文宣數量 - 簽到流程 - 問卷回收方式

    原則上應提前到場，確認： - 場地開放狀況 - 投影、音響、麥克風 - 桌椅擺設 - 背板、立牌、文宣區 - 報到區與簽到表 - QR code 是否可正常掃描

    2. 進行報到與資料發放¶

    如有報到程序，應確認： - 簽到 - 資料領取 - 宣導品發放 - 現場引導

    3. 宣導執行¶

    宣導執行時，應注意： - 主題清楚 - 內容與對象相符 - 用語不過度艱深 - 有需要時補充中心聯繫方式、服務對象與服務內容 - 活動中可視需要引導加入官方 LINE、索取 DM 或留後續聯繫資訊

    監護或輔助宣告業務行政流程¶

    監護或輔助宣告業務屬本中心重要且具司法性質之專案業務，涉及縣府在案追蹤、法院調查訪視、教育訓練與宣導等多項工作。此業務除需具備法律與評估敏感度外，行政流程亦須明確，才能確保收案、派案、訪視、報告審查、寄出與後續核銷均能依時完成。 目前監輔宣業務可區分為三部分： - 1. 縣府監輔宣在案追蹤。 - 2. 監輔宣案件調查訪視。 - 3. 監輔宣教育訓練、外聘督導與宣導活動。

    二、步驟／流程¶

    1. 收案¶

    案件來源通常由縣府以紙本或電子檔交由本中心收件。收到後，不可只口頭轉知，應進入正式收發流程。

    2. 行政收發與建檔¶

    行政收到後，應先完成： - 1. 收件登錄。 - 2. 基本建檔。 - 3. 通知進入篩派案流程。 相關收發資料夾可參考： - FS 內部伺服器指定資料夾（公開版不列示完整路徑，請依內部公告或督導提供版本查找）

    3. 篩派案¶

    篩派案時應： - 1. 依公文內容初步釐清案情。 - 2. 致電聲請人確認本案目前概況。 - 3. 告知本中心後續將進行家庭訪視。 - 4. 釐清目前是否已進行醫療鑑定。 - 5. 依輪序表聯繫訪員。 - 6. 以公務信箱派案並附上公文、表格與參考範本。

    4. 案件管理與追蹤¶

    案件派出後，負責追蹤者應： - 1. 追蹤訪員案件處理狀況。 - 2. 協助處理疑難問題。 - 3. 定期追蹤尚未回覆或進度落後案件。

    5. 報告收件¶

    訪員完成調查訪視後，行政負責收件、歸檔，並通知督導進行審閱。

    6. 報告審查¶

    目前原則為： - 1. 林彥宇督導案件，由黃明珠督導審查。 - 2. 其他案件主要由林彥宇督導審核。 - 3. 審核完成後，告知行政確定版內容。

    7. 報告寄出與後續核銷¶

    行政確認報告後，應： - 1. 擬公文。 - 2. 紙本送出。 - 3. 註記送出日期。 - 4. 辦理後續核銷。 目前本中心成員自行處理案件原則上不核銷；若屬委託其他方案訪員案件，才另行進行核銷。

    監護或輔助宣告業務行政流程¶

    一、目的¶

    本節整理監護或輔助宣告業務在中心內的行政流程，協助同仁從收件、建檔、篩派案、訪視、報告收件、審查、寄出到後續追蹤，都有一致的辦理脈絡。

    二、業務定位¶

    監護或輔助宣告業務並不是單一一份報告而已，而是一整套行政與專業流程。 依中心現行工作內容，至少可分為三部分： - 1. 在案追蹤 - 2. 調查訪視 - 3. 教育訓練、外聘督導與宣導活動 因此承辦此業務時，不能只看單次訪視或單次送件，而要把整體流程、資料管理與後續追蹤一起看。

    三、整體流程總覽¶

    監護或輔助宣告業務原則上可整理為下列步驟： - 1. 收件 - 2. 建檔 - 3. 篩派案 - 4. 主責調查訪視 - 5. 撰寫與收受報告 - 6. 內部審查與修正 - 7. 正式寄出或回送 - 8. 在案追蹤或後續行政處理 - 9. 核銷、成果與資料歸檔

    四、詳細流程¶

    步驟一：收件¶

    案件來源可能來自法院、公文、收發系統或其他正式移送。 收件時應先確認： - 案件來源 - 收件日期 - 案件基本資料 - 是否已有既有資料或舊案紀錄 - 是否需先登記於收發系統 若為正式來文或正式送件，應依中心既定收發流程辦理，不可僅口頭轉交。

    步驟二：建檔¶

    收件後，應建立對應案件資料，至少包含： - 案件基本資料 - 收件來源 - 指派狀態 - 應完成期限 - 對應資料夾或電子檔位置 建檔的目的，在於後續可追蹤案件目前在哪一階段，而不是等到報告快到期才回頭補整理。

    步驟三：篩派案¶

    建檔後應進行篩派案。 篩派案時，應考量： - 區域與案件分布 - 現行案量 - 主責熟悉度 - 時效性 - 是否涉及特殊情況或需督導特別協調 如有特殊案件、複雜家屬互動、法院期限緊迫或需跨系統判斷之情形，應先與督導討論後再派。

    步驟四：主責調查訪視¶

    主責接案後，應依業務需要進行： - 個案訪視 - 家屬或相關人訪談 - 功能與支持狀況了解 - 權利保障與實際生活處境評估 - 需要時蒐集佐證資料 此階段的重點，不只是完成形式訪視，而是協助形成可供後續判斷之完整事實基礎。

    步驟五：撰寫與收受報告¶

    主責完成訪視與資料蒐集後，應依中心規範撰寫報告。 報告收回後，應確認： - 是否已完成 - 格式是否正確 - 內容是否足夠支撐判斷 - 是否有缺頁、漏簽、漏附件等問題

    步驟六：內部審查與修正¶

    報告完成後，不宜直接寄出。 原則上應經內部審查，確認： - 內容是否清楚 - 判斷是否一致 - 用語是否適切 - 是否回應案件核心問題 - 是否仍需補件、補問或補修 若審查後有缺漏，應退回主責修正。

    步驟七：正式寄出或回送¶

    經確認無誤後，再依案件性質正式寄出、函覆或送回。 若涉及法院、正式公文或對外送件，應依中心收發與用印規則辦理。

    步驟八：在案追蹤與後續行政處理¶

    部分案件不是寄出就完全結束，仍可能涉及： - 後續追蹤 - 補件 - 回文 - 另案處理 - 與監護輔助業務相關之持續聯繫 因此主責或承辦仍須確認是否需保留後續追蹤狀態。

    步驟九：核銷、成果與資料歸檔¶

    如該業務同時涉及： - 教育訓練 - 外聘督導 - 宣導活動 - 相關成果資料 則後續還需完成成果、核銷與資料歸檔，不可只停在個案報告寄出。

    五、資料管理重點¶

    監護或輔助宣告業務應特別注意資料可追溯性。 至少要做到： - 1. 收件可追 - 2. 派案可追 - 3. 報告版本可追 - 4. 寄出狀態可追 - 5. 後續追蹤可追 若資料散落於個人桌面、訊息視窗或未命名檔案中，後續非常容易失去行政脈絡。

    家庭微旅行FUN出門運作原則與推薦制度¶

    一、目的與定位¶

    「家庭微旅行 FUN 出門」為苗栗縣身心障礙者服務中心（第一區）115 年度之家庭支持活動，同時也是一項可直接納入個案處遇的服務資源。此資源不是一般性公開招募活動，而是由主責社工依個案處遇需要進行評估、邀請與推薦，透過小團制、低門檻、可被支持的外出方式，協助服務對象及家庭照顧者跨出家門，增加社會參與、家庭互動與正向經驗。 本資源之主要用途如下： - 1. 增進個案與家庭照顧者之正向互動。 - 2. 增進主責社工與案家之關係與服務配合度。 - 3. 提供個案實際社會參與經驗。 - 4. 作為社會參與與充權處遇資源。 - 5. 作為家庭互動觀察、評估與後續處遇之延伸場域。 因此，主責社工不得將本資源視為單純活動通知，而應視為一項有評估、有安排、有陪同、有觀察與有後續追蹤的處遇工具。

    二、招募原則¶

    （一）現行正式做法¶

      1. 本活動採「主責推薦、邀請制」。
      1. 對外不公開廣招。
      1. Google 表單僅作為資料蒐集工具，不代表公開報名。
      1. 是否邀請、邀請誰、是否適合參與，仍由主責社工評估並與督導討論。

    （二）不公開廣招之原因¶

      1. 本活動需由主責社工事前評估個案及家屬之參與適切性。
      1. 本活動涉及保險、交通、飲食、無障礙支持、情緒與醫療風險掌握，不適合無前置評估即直接納入。
      1. 本活動之核心不只是出遊，而是透過陪同與支持，促進處遇與服務連結。
      1. 活動當日原則上由推薦之主責社工陪同參與，因此需依人力與個案狀況審慎安排。

    三、適用對象¶

    （一）主要對象¶

      1. 本中心開案服務中，且目前於社區生活之身心障礙者。
      1. 因障礙、經濟、家庭困頓、照顧安排、缺乏交通工具等因素，難以外出進行社會參與之身心障礙者。
      1. 家庭支持系統薄弱、疏離，或長期缺乏外出經驗與社會參與經驗之案家。
      1. 經主責人員評估並推薦者。

    （二）參與型態¶

      1. 個案可單獨參與。
      1. 家庭照顧者可單獨參與。
      1. 個案與家庭成員亦可共同參與。
      1. 單戶實際參與人數，可依個案需要由主責社工彈性評估，不以單一模式限制。

    四、適用條件與原則¶

    以下屬判斷原則，非絕對限制，但仍應由主責與督導綜合判斷： - 1. 距活動地點 15 分鐘內車程者為佳。 - 2. 距活動地點 30 分鐘內車程者可評估。 - 3. 不宜重複參與，但如有特殊處遇考量，可與督導討論後評估。 - 4. 以真正難以出門、社會參與不足、外出需要支持者優先。 - 5. 若個案已有長期穩定外出能力，原則上不優先使用本資源。

    五、主責社工之角色與責任¶

    （一）前期工作¶

      1. 主責社工評估案家是否適合參與。
      1. 由主責社工主動邀請，而非被動等待案家自行報名。
      1. 蒐集案家外出可能遇到之阻礙，例如：
    • 交通不便
    • 無人陪同
    • 視覺障礙
    • 聽語障礙
    • 情緒或行為風險
    • 飲食限制
    • 行動輔具需求
      1. 將案家可能遇到之問題帶回團隊討論，尋求可行之解決方案。
      1. 協助案家完成活動資料填寫、保險資料與需求資料整理。
      1. 於活動前完成行前提醒，並確認案家已知悉時間、地點、集合方式與注意事項。

    （二）活動當日角色¶

      1. 推薦者即為天然當天工作人員。
      1. 主責社工之主要任務，是陪同自己服務之案家。
      1. 若推薦主責因故無法出席，應由職代陪同參與，不可無人陪同。
      1. 主責社工需在活動中觀察個案參與狀況、家庭互動、風險點與後續可延伸服務方向。

    （三）活動後角色¶

      1. 協助回收或確認滿意度調查表完成填寫。
      1. 提供本次活動之回饋與精進事項。
      1. 視需要將活動觀察納入個案處遇與後續追蹤。
      1. 持續與服務對象及家屬聯繫，維持服務延續性。

    六、年度服務指標¶

    （一）整體指標¶

      1. 全年共辦理 6 場次。
      1. 年度總服務目標為 90 人次。

    （二）主責人員指標¶

      1. 115 年原在職主責人員：每人全年 9 人次，不可重複。
      1. 115 年 5 月後到職者：每人 4 人次，不可重複。
      1. 人次以實際參與者計算，不得以同一人重複累計為達標方式。
      1. 每場次每位主責人員維持 1 至 2 人次之基本出席量，以利全年進度平均推進。

    七、可提供之支持方式¶

    若案家因外出困難而影響參與，主責社工應協助辨識問題並與團隊討論解法，例如： - 1. 交通不便者，可協助安排復康巴士。 - 2. 可由社工教導使用公車等公共運輸方式。 - 3. 如前述方式仍不可行，可評估提供計程車車資。 - 4. 視覺障礙、聾啞等個案，可視需求安排適當支持，例如手語或其他溝通協助。 - 5. 其他與外出有關之阻礙，均可在第一次討論會及行前會中提出並討論。

    八、重要提醒¶

      1. 本活動為處遇資源，不得廣發、不宜公開招募。
      1. Google 表單與後台試算表均屬資料蒐集與確認工具，非公開報名平台。
      1. 主責社工應確實評估適配性，不宜僅為完成指標而推薦不適切對象。
      1. 活動安排之重點不是「把人帶出來」而已，而是要確保能安全外出、順利參與，並於過程中形成服務與評估價值。

    三、活動前流程¶

    （一）規劃期¶

    由督導與小行政先進行整體規劃，內容包含： - 1. 場地確認與活動內容規劃。 - 2. 初步交通方式盤點。 - 3. 經費與執行可行性確認。 - 4. 前次活動回饋與修正事項整理。 - 5. 招募方式、表單、流程及文宣準備。 主責人員於規劃期可提供場地意見、需求觀察與實務修正意見。

    （二）招募與推薦期¶

      1. 主責社工依個案處遇需求邀請案家。
      1. 主責社工蒐集外出困難與需求，帶回討論。
      1. 主責社工協助填寫推薦資料、保險資料、交通需求、飲食限制等資訊。
      1. 小行政李芸嫻收受推薦資料，確認是否完整。
      1. 督導審核推薦家戶與整體安排。

    （三）報名截止與保險資料¶

      1. 各場次最晚於活動當週星期一完成報名資料。
      1. 保險清冊務必於活動當週星期一完成確認，原因為需替案家辦理保險。
      1. 若個案參與需要任何協助或特殊支持，應最晚於同一截止日前提出。
      1. 小行政彙整保險資訊後，交由行政黃梅辦理投保。

    四、第一次討論會¶

    （一）辦理目的¶

    第一次討論會為各場次之前置工作會議，主要功能為： - 1. 盤點目前推薦個案與家戶狀況。 - 2. 確認每戶外出阻礙與解決方案。 - 3. 討論交通方式、陪同安排、飲食需求及特殊支持。 - 4. 確認是否有需督導協助或團隊支援之事項。 - 5. 確保主責社工於活動前已知悉各案家之重點需求。

    （二）會議時間¶

    原則上於活動當週星期一下午 13:30 開始。 第 2 場因當日白天既有教育訓練，改於 115/05/19（二）辦理。

    （三）主責於第一次討論會需帶入之資訊¶

      1. 案家是否確定參與。
      1. 個案與家屬參與人數。
      1. 交通方式與交通困難。
      1. 飲食限制、用藥、輪椅、輔具、移位、情緒或醫療風險。
      1. 是否需要手語或其他溝通協助。
      1. 是否需補助計程車、復康巴士或其他支持措施。
      1. 主責當天是否可出席；若不可，職代為誰。

    五、行前會¶

    （一）辦理目的¶

    行前會為活動前最後確認會議，主要在於： - 1. 再次確認參與名單。 - 2. 確認交通安排與集合方式。 - 3. 確認每位主責陪同家戶。 - 4. 確認機動成員與攝影成員。 - 5. 確認場地動線、醫藥箱、設備與物資。 - 6. 確認滿意度調查表、照片回收與當日紀錄方式。

    （二）行前會日期¶

    依各場次表定日期辦理，原則上為活動前一天。

    六、活動當日角色分工¶

    （一）主責人員¶

      1. 推薦者即為天然當天工作人員。
      1. 主責人員任務為陪同自己服務之案家。
      1. 確認案家安全、互動、活動參與及突發需求。
      1. 協助案家理解流程、集合與返程安排。
      1. 協助案家填寫滿意度調查表。

    （二）督導與機動組¶

      1. 現行主要由林彥宇督導與機動組成員處理各種問題與突發狀況。
      1. 至少需配置 1 名機動成員。
      1. 工作包括突發狀況處理、協助交通與現場協調、提醒拍攝及表單回收、支援各主責人員。

    （三）攝影紀錄成員¶

      1. 至少需有 1 名攝影紀錄協助成員。
      1. 負責拍攝每個家戶與參與者之照片。
      1. 負責拍攝開場照片、大合照及活動照片。
      1. 協助後續成果整理所需之影像紀錄。

    （四）職代原則¶

    若推薦主責因故無法出席，應由職代陪同參與，不得無主責或無陪同人員即讓案家自行參與。

    七、活動後工作¶

      1. 主責人員確認滿意度調查表是否完成。
      1. 各組務必協助個案填寫滿意度調查表。
      1. 小行政回收回饋、照片、人次資料與名冊資料。
      1. 督導與小行政整理成果、檢討與下次優化事項。
      1. 主責將活動觀察納入後續服務追蹤。

    八、重要提醒¶

      1. 第一次討論會與報名截止日，務必視為保險清冊與參與安排之最晚期限。
      1. 未完成資料、未提出需求、未確認交通方式者，容易影響活動安全與投保安排。
      1. 活動日所有問題及狀況，原則上由林彥宇督導與機動組協助處理。
      1. 主責最重要的任務，是陪同自己的案家，而不是兼做全場機動。
      1. 滿意度調查表之填寫，不得遺漏。

    七、素材收繳時間¶

    為避免活動快到才追素材，採下列節奏：

    如身障微旅、一家心聚、一般宣導活動，原則上應於活動前至少 3 週提供預告素材。 若有成果文，應於活動後儘速提供照片、摘要與成果重點。

    3. 月報數據¶

    社群小行政應於每月共識會議前完成： - LINE 好友數統整 - FB 追蹤數與互動數據整理 - 待處理訊息盤點

    八、月度管理指標¶

    目前可作為共識會議與小行政追蹤的基本指標包括： - 1. 素材收繳完成率 - 2. 發文排程完成率 - 3. LINE 好友數 - 4. FB 追蹤數 - 5. 貼文瀏覽或互動數據 - 6. 各主題年度應達量是否如期推進

    獨居且行動不便身障者專案關懷任務¶

    一、目的¶

    本節整理 115 年度「獨居且行動不便」身心障礙者專案關懷任務之清查目標、分級要求、服務內容、回報時程與督導提醒，協助同仁在接獲專案名冊後，能依一致方式完成清查與回報。

    二、任務來源與定位¶

    本任務依苗栗縣政府 115 年 2 月 5 日函文辦理，屬轄內高風險獨居身障者之專案清查與即時關懷工作。 核心目的在於： - 針對中央系統篩選之「獨居且行動不便」身障個案進行地毯式清查 - 確認其實際居住現況 - 及早連結福利、輔具、物資與照顧支持 - 更新聯絡資料，降低失聯與風險累積

    三、強制分級基準¶

    本專案最重要的規則之一，是名冊內所有個案： - 至少應列為第 3 級 若家訪或後續查核發現風險更高，則應依實際狀況再調升為： - 第 1 級 - 或第 2 級 也就是說，這批清冊個案原則上不可直接列為第 4 級。

    四、服務頻率¶

    若列為第 3 級，至少應維持： - 每半年 1 次電話訪問 如個案風險高於一般第 3 級，應再依實際需要提高追蹤與服務強度。

    五、專案重點工作內容¶

    主責社工至少應完成下列查核：

    1. 確認是否確實居住於轄區¶

    不能只看名冊資料，仍應確認： - 是否仍住在本轄 - 是否已遷出 - 是否已死亡 - 是否已入住機構

    2. 評估居家生活安全¶

    至少應注意： - 是否有跌倒風險 - 是否有居家環境安全問題 - 是否有無法自行取得食物、藥物或日用品之風險 - 是否有緊急求援困難

    3. 評估是否有即時資源需求¶

    包括但不限於： - 物資 - 輔具 - 照顧人力 - 福利連結 - 交通支持

    4. 更新聯繫方式¶

    需確認並更新： - 個案最新聯絡方式 - 緊急聯絡人 - 其他可聯絡支持者資訊

    六、行政回報要求¶

    1. 清查期限¶

    應於： - 115 年 5 月 10 日前 完成清查與回報。

    2. 回報格式¶

    應依公文附件之： - 身心障礙者個別清查結果表 填寫。

    3. 回報對象¶

    收件窗口為： - 社會處身障科 李文琦社工

    4. 回報方式¶

    應將電子檔寄送至： - wenchi_li@ems.miaoli.gov.tw

    七、督導與主責提醒¶

    依公文核章備註，中心督導應於期限前提醒同仁： - 完成電訪 - 落實列冊人數回復 - 確認清查狀態 若主責發現個案： - 已遷出 - 已死亡 - 已入住機構 也應於報表中詳實註記，以利資料更新。

    八、主責社工的工作節奏¶

    接到清冊後，應依下列順序處理： - 1. 先確認名冊資料 - 2. 先做電話聯繫與基本查核 - 3. 視需要家訪或進一步聯繫 - 4. 判斷維持第 3 級或調升第 1／2 級 - 5. 完成清查結果表 - 6. 於期限前回報 - 7. 後續依分級維持追蹤

    九、與一般三級案件的差異¶

    雖然本專案最低為第 3 級，但其特殊性在於： - 這不是一般自由判斷後才列 3 級的案件 - 而是專案名冊中的個案，原則上即以第 3 級起跳 - 後續再依風險調升，而非往下放 因此，主責在處理時，應有專案名冊意識與列冊回報意識。

    十、常見錯誤¶

    • 收到名冊卻未實際聯繫
    • 直接放成第 4 級
    • 沒更新緊急聯絡方式
    • 發現遷出、死亡或入住機構卻沒註記
    • 清查後沒回報結果表
    • 只做一次聯絡，沒有後續依分級持續追蹤

    十一、注意事項¶

    • 專案名冊內個案最低為第 3 級。
    • 家訪後若風險更高，應調升而非維持低強度。
    • 回報期限與表單格式均屬正式行政要求，不可省略。
    • 名冊涉及個資，應妥善保管。
    • 本篇應與案件分級、獨居行動不便類型與不宜降為 4 級的規則一併閱讀。

    十二、相關主題¶

    • 案件分級與訪視頻率
    • 不得降為四級案件類型與高風險家庭評估
    • 開案前查核與開案須知
    • 派案輪序與主責原則

    宣導品核稿、揭示字樣與核銷注意事項¶

    一、目的¶

    本節整理 115 年度宣導品、手冊、課程表與其他文宣製作前應注意的核稿流程、必要揭示字樣與核銷要求，避免文宣做完後才發現漏字、漏 Logo，影響後續核銷。

    二、這篇適用哪些內容¶

    本篇所稱宣導品，包含但不限於： - 海報 - 手冊 - 課程表 - 活動簡章 - DM - 宣導摺頁 - 講座文宣 - 其他需列印或對外公開之宣導品

    三、核稿的基本原則¶

    依 115 年 3 月業務聯繫會議佈達，所有文宣品於製作前，原則上應： - 先送身障服務科核稿審閱 目的在於避免因漏記必要標示或格式錯誤，導致後續核銷困難。

    四、必須揭示的文字與圖示¶

    凡屬本計畫下製作之文宣、手冊、課程表或宣導品，應至少揭示： - 1. 苗栗縣政府委託 - 2. 苗栗縣公益彩券盈餘分配基金補助 - 3. 公益彩券 Logo 以上三者原則上均不可漏。 若漏記，後續核銷極可能受影響。

    五、實務操作¶

    1. 一開始就把字樣放進版型¶

    不要等設計快完成才補字。 應於文宣初版就先把： - 委託字樣 - 補助字樣 - Logo 放入固定版位。

    2. 核稿前先自我檢查¶

    送核稿前，應先自行檢查： - 日期、時間、地點是否正確 - 主辦／承辦／協辦單位是否完整 - 必要揭示字樣是否已放 - 公益彩券 Logo 是否已置入 - 是否有錯字與格式問題

    3. 留下核稿版本¶

    應保留： - 初稿 - 送核版本 - 修正版 - 最終定稿 以利後續對照與必要時追溯。

    六、與文宣審核區的關係¶

    宣導素材實務上常會先放於文宣審核區，再進行後續發布或印製。 因此同仁在製作宣導品時，也應同步注意： - 資料夾命名是否清楚 - 是初稿、送審稿還是定稿 - 是否已完成科內核稿 - 是否可直接進入印製或發布

    七、哪些情況最容易出錯¶

    最常見的錯誤包括： - 只顧版面設計，忘了委託與補助揭示 - 有放單位名稱，卻沒放公益彩券 Logo - 先印了才送核稿 - 同一文宣多版本流傳，最後不知道哪份是核定版 - 把社群圖卡和正式印刷文宣混在一起，導致核稿要求搞混

    八、核銷意識¶

    宣導品不是做完就好，還要能核銷。 因此從一開始就要有核銷意識： - 1. 這份文宣是否屬正式核銷項目 - 2. 是否符合計畫揭示要求 - 3. 是否已先送審 - 4. 是否保留可供核銷的定稿版本 - 5. 是否留有後續成果或發放佐證

    九、常見流程¶

    可依下列順序進行： - 1. 確認用途與主題 - 2. 套用既有版型 - 3. 先放必要揭示字樣與 Logo - 4. 完成初稿 - 5. 送身障服務科核稿 - 6. 修正後定稿 - 7. 再印製或正式發布 - 8. 保留定稿與核銷資料

    十、注意事項¶

    • 正式文宣製作前先送核稿，是 115 年的明確要求。
    • 「苗栗縣政府委託」、「苗栗縣公益彩券盈餘分配基金補助」及公益彩券 Logo 不可漏。
    • 圖卡、手冊、課程表、簡章等均應注意。
    • 若後續主管機關對揭示格式有更新，應同步修正本篇。
