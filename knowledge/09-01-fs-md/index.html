    FS內部伺服器與常用資料夾基本使用¶

    編號：09-01 狀態：正式 章節：第9章 工具與系統操作

    一、目的¶

    FS 為中心日常工作最核心的內部伺服器。各類表單、公版、教材、核銷資料、活動資料、用印資料、系統申請表、成果檔案與行政檔案，多數皆放置於 FS。新人進入中心後，應先熟悉 FS 的基本位置與常用操作方式，避免找不到表單、使用錯誤版本，或因檔案放錯位置而影響工作銜接。

    二、FS 基本位置¶

    中心常用主路徑如下：

    FS 內部伺服器指定資料夾（公開版不列示完整路徑，請依內部公告或督導提供版本查找）

    同仁可由此路徑再進入各資料夾，依工作需要查找： - 庶務性工作 - 系統申請表單 - 交通費核銷 - 家庭支持活動 - 收發系統 - 用印申請 - 教材與簡報 - 各方案或各章節相關附件

    三、FS 的主要用途¶

    FS 並不只是檔案倉庫，而是中心正式工作資料的共用空間。實務上可分成幾類：

    1. 表單與公版¶

    例如： - 系統申請表 - 物資領據 - 用印申請書 - 核銷總明細表 - 活動資料表單 - 成果與回饋彙整表

    2. 教材與操作指引¶

    例如： - TimeTree 教學簡報 - 身心障礙個案管理系統教材 - iTalk 操作說明 - IP Phone 撥號手冊 - 官方 LINE 使用指南

    3. 行政與核銷資料¶

    例如： - 每月交通費登打檔案 - 用印申請紀錄 - 活動成果照片與報告 - 各類核銷附件

    4. 活動與專案資料夾¶

    例如： - 家庭微旅行 FUN 出門 - 一家心聚系列活動 - 家庭支持活動 - 精進專業服務計畫相關資料

    四、常見使用方式¶

    1. 查找固定公版¶

    若工作內容有固定表單，應優先到對應資料夾查找最新版，不建議自行沿用舊檔或桌面暫存版本。

    2. 依既有命名邏輯保存¶

    若需新增檔案或更新資料，應依中心既有命名方式辦理，保持團隊查找一致性。若尚不確定命名規則，應先詢問督導、行政或小行政主責。

    3. 注意是否為共用主檔¶

    部分 Excel 或總表檔案為共用主檔，編輯前應確認： - 是否可以直接改動 - 是否應另存新檔 - 是否已有當月或當年度專用版本

    4. 注意資料夾用途¶

    同一份文件若應歸檔於活動資料夾、行政資料夾或核銷資料夾，位置不可混放。放錯位置會造成後續查找、成果彙整與核銷困難。

    五、常用資料夾示例¶

    1. 系統申請表公版¶

    FS 內部伺服器指定資料夾（公開版不列示完整路徑，請依內部公告或督導提供版本查找）

    2. 家庭微旅行資料夾¶

    FS 內部伺服器指定資料夾（公開版不列示完整路徑，請依內部公告或督導提供版本查找）

    3. 用印相關資料夾¶

    依用印申請與收發系統實際資料夾位置辦理。

    六、VBA 無法開啟巨集時的處理¶

    若開啟 FS 內含巨集的 Excel 檔案時，出現巨集遭封鎖或無法啟用情形，可依下列方式處理：

    1. 開啟「執行」視窗。
    2. 輸入：inetcpl.cpl
    3. 切換至「安全性」頁籤。
    4. 點選「近端內部網路」→「網站」→「進階」。
    5. 新增：FS 內部伺服器（公開版不列示連線位址）
    6. 加入網站清單後再重新開啟檔案。

    如仍無法使用，應請熟悉電腦設定之同仁或相關承辦協助處理。

    七、注意事項¶

    • FS 內多為正式工作檔案，不建議任意刪除、覆蓋或改檔名。
    • 若需修改共用檔，應先確認是否會影響他人作業。
    • 含個資、案家資訊、評估資料、回饋資料之文件，仍應依保密原則使用。
    • 若資料夾中已有既定版本，應避免自行另開多個命名相近的版本造成混亂。
    • 若發現資料夾位置異動、路徑失效或公版缺漏，應回報督導、行政或主責小行政。
