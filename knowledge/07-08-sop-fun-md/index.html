    家庭微旅行FUN出門活動協作流程¶

    編號：07-08 狀態：正式 章節：第7章 業務流程與專案SOP

    一、流程定位¶

    本篇說明家庭微旅行 FUN 出門之整體協作流程，包含活動前之推薦、第一次討論會、資料蒐集、保險清冊、行前會、活動當日分工及活動後回收與追蹤。此流程之核心原則為：主責社工推薦、主責社工陪同、團隊共同協助排除外出障礙，並將活動作為個案與家庭支持之處遇資源。

    二、年度場次、報名截止、第一次討論會與行前會日期¶

    原則說明：
    1. 報名截止最晚為活動當週星期一；若遇假日則提前至前一工作日。
    2. 第一次討論會原則上安排於活動當週星期一，通常自下午 13:30 開始。
    3. 由於活動需替案家辦理保險，保險清冊務必於活動當週星期一完成確認。
    4. 行前會原則上安排於活動前一天；若遇假日則提前至前一工作日。
    5. 本年度第 2 場因 5/18 已有教育訓練，第一次討論會調整至 5/19 辦理。
    6. 第 1 場日期雖已過，但仍保留於手冊中作為完整流程記錄。

    三、活動前流程¶

    （一）規劃期¶

    由督導與小行政先進行整體規劃，內容包含： 1. 場地確認與活動內容規劃。 2. 初步交通方式盤點。 3. 經費與執行可行性確認。 4. 前次活動回饋與修正事項整理。 5. 招募方式、表單、流程及文宣準備。

    主責人員於規劃期可提供場地建議、需求觀察與實務修正意見。

    （二）招募與推薦期¶

    1. 主責社工依個案處遇需求邀請案家。
    2. 主責社工蒐集外出困難與需求，帶回討論。
    3. 主責社工協助填寫推薦資料、保險資料、交通需求、飲食限制等資訊。
    4. 小行政李芸嫻收受推薦資料，確認是否完整。
    5. 督導審核推薦家戶與整體安排。

    （三）報名截止與保險資料¶

    1. 各場次最晚於活動當週星期一完成報名資料。
    2. 保險清冊務必於活動當週星期一完成確認，原因為需替案家辦理保險。
    3. 若個案參與需要任何協助或特殊支持，應最晚於同一截止日前提出。
    4. 小行政彙整保險資訊後，交由行政黃梅辦理投保。

    四、第一次討論會¶

    （一）辦理目的¶

    第一次討論會為各場次之前置工作會議，主要功能為： 1. 盤點目前推薦個案與家戶狀況。 2. 確認每戶外出阻礙與解決方案。 3. 討論交通方式、陪同安排、飲食需求及特殊支持。 4. 確認是否有需督導協助或團隊支援之事項。 5. 確保主責社工於活動前已知悉各案家之重點需求。

    （二）會議時間¶

    原則上於活動當週星期一下午 13:30 開始。
    第 2 場因當日白天既有教育訓練，改於 115/05/19（二）辦理。

    （三）主責於第一次討論會需帶入之資訊¶

    1. 案家是否確定參與。
    2. 個案與家屬參與人數。
    3. 交通方式與交通困難。
    4. 飲食限制、用藥、輪椅、輔具、移位、情緒或醫療風險。
    5. 是否需要手語或其他溝通協助。
    6. 是否需補助計程車、復康巴士或其他支持措施。
    7. 主責當天是否可出席；若不可，職代為誰。

    五、行前會¶

    （一）辦理目的¶

    行前會為活動前最後確認會議，主要在於： 1. 再次確認參與名單。 2. 確認交通安排與集合方式。 3. 確認每位主責陪同家戶。 4. 確認機動成員與攝影成員。 5. 確認場地動線、醫藥箱、設備與物資。 6. 確認滿意度調查表、照片回收與當日紀錄方式。

    （二）行前會日期¶

    依各場次表定日期辦理，原則上為活動前一天。

    六、活動當日角色分工¶

    （一）主責人員¶

    1. 推薦者即為天然當天工作人員。
    2. 主責人員任務為陪同自己服務之案家。
    3. 確認案家安全、互動、活動參與及突發需求。
    4. 協助案家理解流程、集合與返程安排。
    5. 協助案家填寫滿意度調查表。

    （二）督導與機動組¶

    1. 現行主要由林彥宇督導與機動組成員處理各種問題與突發狀況。
    2. 至少需配置 1 名機動成員。
    3. 工作包括突發狀況處理、協助交通與現場協調、提醒拍攝及表單回收、支援各主責人員。

    （三）攝影紀錄成員¶

    1. 至少需有 1 名攝影紀錄協助成員。
    2. 負責拍攝每個家戶與參與者之照片。
    3. 負責拍攝開場照片、大合照及活動照片。
    4. 協助後續成果整理所需之影像紀錄。

    （四）職代原則¶

    若推薦主責因故無法出席，應由職代陪同參與，不得無主責或無陪同人員即讓案家自行參與。

    七、活動後工作¶

    1. 主責人員確認滿意度調查表是否完成。
    2. 各組務必協助個案填寫滿意度調查表。
    3. 小行政回收回饋、照片、人次資料與名冊資料。
    4. 督導與小行政整理成果、檢討與下次優化事項。
    5. 主責將活動觀察納入後續服務追蹤。

    八、重要提醒¶

    1. 第一次討論會與報名截止日，務必視為保險清冊與參與安排之最晚期限。
    2. 未完成資料、未提出需求、未確認交通方式者，容易影響活動安全與投保安排。
    3. 活動日所有問題及狀況，原則上由林彥宇督導與機動組協助處理。
    4. 主責最重要的任務，是陪同自己的案家，而不是兼做全場機動。
    5. 滿意度調查表之填寫，不得遺漏。
