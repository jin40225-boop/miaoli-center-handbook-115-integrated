    出缺勤與請假流程¶

    編號：03-01 狀態：正式 章節：第3章 人事差勤與辦公室規範

    一、正文說明¶

    本篇用於說明本中心同仁日常出缺勤、請假與人事考勤的基本做法。中心採彈性工作模式，不設打卡制度，運作基礎是互信、自律與即時留下工作軌跡。因此，請假不是只有口頭告知即可，而是必須依中心既定程序，同時完成「向督導請示」與「系統登錄」兩個動作，讓督導、同仁與行政都能掌握出勤狀況，避免工作交接落空、值班銜接中斷，或兩位督導資訊不同步。

    本篇重點放在中心現行實務流程，也就是同仁實際每天會用到的請假程序、TimeTree 登錄方式、何時需要請假單，以及按月考勤資料填寫。若涉及法定假別定義、工時、特休與請假權益，請再搭配本章其他文章及協會工作規則一併閱讀。

    二、步驟／流程¶

    1. 先判斷自己是否需要請假或更新出勤異動¶

    凡有下列情況之一，應主動處理出勤異動： 1. 當日無法出勤。 2. 需請假半日、數小時或整日。 3. 行程有臨時異動，需要讓督導與同仁知道目前出勤安排。 4. 需要留下後續考勤與人力調度依據。

    2. 先在 LINE 三人社群向督導請示¶

    中心目前請假原則，不以私下口頭告知為完成。實務上，若僅很早以前口頭講過，容易造成兩位督導掌握資訊不同步，因此正式流程是： 1. 先到指定的 LINE 三人社群請示督導。 2. 說明請假日期、時間與性質。 3. 如有必要，可簡要說明原因；若理由較私密，可私下與督導說明，但群組中仍須留下日期與時間。 4. 確認已獲任一督導核可後，才能進行下一步系統登錄。

    3. 再到 TimeTree 登錄行程¶

    督導核可後，應立即至 TimeTree 更新出勤資訊，讓團隊行程維持同步。

    基本做法如下： 1. 開啟中心共用 TimeTree 行事曆。 2. 建立對應行程。 3. 行程名稱依中心規則登錄。 4. 填寫請假日期、起訖時間。 5. 視需要補充說明。 6. 完成後，務必再點選公告或留言標記，留下已完成登錄的軌跡，讓督導知道本次請假已完成系統更新。

    4. 依格式正確登錄請假資訊¶

    請假或出勤異動時，應依中心既定命名方式登打。若為一般請假，可至少清楚寫出「性質－姓名」。若為外出或其他行程，應再補充個案名稱與前往地區，方便同仁辨識。

    可參考以下寫法： - 請假－彥宇 - 補休－彥宇 - 外訪－姓名－個案名稱－前往地區

    若該行程涉及職務代理、半日請假或特殊狀況，也可依中心當下最新公告格式補充標示。

    5. 視情況判斷是否須填寫實體請假單¶

    不是所有請假都要填實體請假單。依目前中心做法： - 短期、臨時性請假：通常不需填寫實體請假單，但仍須完成督導核可與 TimeTree 登錄。 - 長時間請假、可能涉及扣薪或較正式人事處理者：需填寫實體「請假單」。

    如不確定是否屬於需填寫請假單的情況，應主動詢問督導或行政，不要自行判斷略過。

    6. 按月完成考勤表單填寫¶

    除當次請假程序外，中心也要求同仁配合按月填寫人事考勤相關表單。這是人事管理的一部分，不能因為平常採彈性工時就忽略正式考勤資料。若當月有請假、補休、出勤異動或特殊工時，更應確認是否已反映在月度表單內。

    三、重要知識或判斷原則¶

    1. 中心採彈性工時，不等於沒有規則¶

    本中心雖不採傳統打卡制度，但仍有明確的出缺勤規範。彈性工作模式的前提是：每位同仁都要留下清楚軌跡，讓團隊知道你何時在班、何時請假、何時外出、何時可聯繫。

    2. 請假完成的判準是兩步都完成¶

    請假流程不是只完成其中一項即可，而是： - 已在 LINE 三人社群請示並獲得督導核可。 - 已在 TimeTree 完成登錄並留下標記。

    只有兩者都完成，才算請假程序完成。

    3. 理由可保有隱私，但時間資訊不能省略¶

    若請假原因較私密，可私下與督導說明；但在團隊共用系統中，至少仍應留下日期、時段與請假事實。這是為了維持團隊運作與業務銜接，而非要求公開所有私密內容。

    4. 長時間請假會進入正式人事程序¶

    若請假時間較長、可能涉及扣薪、病假證明、長期調整工時或其他正式人事處理，就不再只是中心日常溝通，而會涉及協會工作規則及正式表單程序，務必提早確認。

    四、注意事項¶

    • 不要只口頭告知督導，未在 LINE 三人社群留下記錄，容易造成資訊不同步。
    • 不要只在 LINE 請假卻未登錄 TimeTree，也不要只登錄 TimeTree 卻未先取得督導核可。
    • 請假登錄完成後，記得在公告或留言區做標記，避免督導未看到。
    • 若行程名稱過於簡略，其他同仁可能無法辨識你的出勤狀況，應盡量依既定格式登錄。
    • 長時間請假、可能扣薪、需要病假證明或涉及法定請假權益時，應主動詢問督導與行政，不要只依過往經驗處理。
    • 每月考勤表單填寫不可遺漏，否則會影響後續人事管理與資料完整性。
